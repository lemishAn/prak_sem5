DataSet находится в той же директории, что и запускаемые программы

Команда 'STAT' - ответ записывается в файл 'statistics.csv' ('statistics_2.csv' в случае http сервера)
Команда не принимает на вход меньше 10 твитов
Команда 'ENTI' - ответ записывается в файл 'enti.json' ('enti_2.json' в случае http сервера)

При обращение к http серверу (адрес после запуска - 127.11.0.1) в случае команды stat делаем запрос к порту 8000, в случае команды enti - 8001.
